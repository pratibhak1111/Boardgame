**Secrets and Environment Variables**  
Created main.yaml file .github/workflows/main.yml  
Github -> Settings → Secrets and variables → Actions  
Click “New repository secret”
Add secrets like:  
SSH_HOST = IP  
SSH_USER = ubuntu  
SSH_KEY = private key
